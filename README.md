# dSphs_ML
data-driven model of the gamma-ray background at dSphs positions according to Fermi-LAT's data.

This code has been used to obtain the results in the paper:
"Dark matter constraints from dwarf galaxies: a data-driven analysis",
Francesca Calore, Pasquale D. Serpico, Bryan Zaldivar
JCAP 10 (2018) 029
1803.05508 [astro-ph.HE]
DOI: 10.1088/1475-7516/2018/10/029

Disclaimer: the code uses datafiles from the public repository of Fermi-LAT's collaboration, which are not included here.
